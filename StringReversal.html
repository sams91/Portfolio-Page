How do you reverse a string?


There are 3 things to understand about reversing a string in JavaScript. The first is there are no built-in functions to reverse strings, which means you can’t simply call a “reverse string” function. Second, it is unlikely you will ever need to do it. Third, although the need to reverse a string rarely comes up people still want to be sure you know how to do it during interviews.

Arrays do have a reverse function. First, we would need to convert our string to an array in the following way:


                    const str = 'this is how we reverse strings';

                    const newStr = str.split('');

                    console.log(newStr);
Here is what the console will show:

0: "t" 1: "h" 2: "i" 3: "s" 4: " " 5: "i" 6: "s" 7: " " 8: "h" 9: "o" 10: "w" 11: " " 12: "w" 13: "e" 14: " " 15: "r" 16: "e" 17: "v" 18: "e" 19: "r" 20: "s" 21: "e" 22: " " 23: "s" 24: "t" 25: "r" 26: "i" 27: "n" 28: "g" 29: "s" length: 30


This splits the string into individual characters, including spaces, and gives each one their own individual index in the array. If we add the reverse method like this: 


                    console.log(newStr.reverse());

The console will show the string in reverse like this:

(30) ['s', 'g', 'n', 'i', 'r', 't', 's', ' ', 'e', 's', 'r', 'e', 'v', 'e', 'r', ' ', 'e', 'w', ' ', 'w', 'o', 'h', ' ', 's', 'i', ' ', 's', 'i', 'h', 't']


Then if we add the join function like this: 


                    console.log(newStr.reverse().join(''));

The console will show this:

sgnirts esrever ew woh si siht


This could be cleaned up by doing it this way: 


                    const str = 'this is how we reverse strings';

                    const newStr = str.split('').reverse().join('');

                    console.log(newStr);

This shows the same result in the console:

sgnirts esrever ew woh si siht


We can also do the same thing using a for loop like this: 


                    const str = 'string reverse with for loop'

                    let newStr = '';

                    for (let i = str.length - 1; i >= 0; i--) {
                    newStr += str[i];
                    }

                    console.log(newStr);


This is what the console will show:

pool rof htiw esrever gnirts


Obviously, there are other ways of doing this. The point is it can be done and it isn’t nearly as complicated as it may sound. Will you see much of this in the wild? Probably not but the knowledge needed to reverse a string can and likely will be applied elsewhere within the coding world.
