What are objects in JavaScript?


Here is an example: 


                    const worker = {
                        name: "bee",
                        job: "hammer swinger",
                        age: 90,
                        favoriteFood: "honey",
                    
                    }
                    
                    console.log(worker.job);
The object is worker. Everything inside the curly brackets {} are properties of the object. Here is what the above example would show in the console:

hammer swinger


Only hammer swinger is shown because the job property of the worker object is being accessed. If we instead access the entire object like this: 


                    console.log(worker);

Then the console will show all the properties like this:

{name: 'bee', job: 'hammer swinger', age: 90, favoriteFood: 'honey'}


You can change properties of the object like this: 


                    const worker = {
                        name: "bee",
                        job: "hammer swinger",
                        age: 90,
                        favoriteFood: "honey",
                    
                    }
                    
                    worker.age = 10
                    
                    console.log(worker.age);

Here we changed the Age property of the worker object from 90 to 10. This is what will show in the console:

10


Now if we access the entire object like this: 


                    console.log(worker);

The console will show this:

{name: 'bee', job: 'hammer swinger', age: 10, favoriteFood: 'honey'}


To delete a property, do this: 


                    const worker = {
                        name: "bee",
                        job: "hammer swinger",
                        age: 90,
                        favoriteFood: "honey",
                    
                    }
                    
                    delete worker.age;
                    
                    console.log(worker);


The console shows this:

{name: 'bee', job: 'hammer swinger', favoriteFood: 'honey'}


Notice the age property is gone.
It is important to note that delete only works with object properties, it will not work on local variables.
Here is an example: 

                    const boss = (function(money) {
                        delete money;
                        return money;
                    }) ("dollars");
                    
                    console.log(boss);


Here we have a function with the variable money as a property of the object boss. Inside the function we are attempting to delete the variable money but delete will not work on a variable, so we are still able to view the object boss in the console without any problems.

Here is what the console will show:

dollars


If a string is used as a property of an object like this: 


                    const worker = {
                        name: "bee",
                        job: "hammer swinger",
                        age: 90,
                        favoriteFood: "honey",
                        "chickens fly fast": "favorite book"
                    }


The string will need to be inside square brackets [ ] in order to be accessed like this: 


                    console.log(worker["chickens fly fast"]);


and the console will show this:

favorite book


Deleting the string property would be done in the same way using the square brackets, [ ].

Now we can add dynamic values to an object like this: 


                    const property = "heavy";
                    const element = "iron";

                    const lifting = {
                    [property]: element,
                    };

                    console.log(lifting);


The console will show this:

{heavy: 'iron'} heavy: "iron" [[Prototype]]: Object


Note the square brackets inside the lifting object. Without them the console would show property: iron instead of heavy: iron.
